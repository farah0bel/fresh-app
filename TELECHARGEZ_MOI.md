## 🚀 LANCER L'APP (5 MINUTES)

```bash

# https://nodejs.org

# 2. Ouvrir un terminal dans le dossier

# 3. Installer les dépendances
npm install

# 4. Lancer l'app
npm run dev

# 🎉 Ouvrir http://localhost:5173
```
