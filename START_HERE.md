## 📥 ÉTAPE 1 : TÉLÉCHARGER (2 minutes)

### **Dans Figma Make :**

1. **Cliquez** sur le bouton `Download` ou `Télécharger` 📥
   - En haut à droite de l'écran
   
2. **Le fichier ZIP se télécharge**
   - Nom : `fresh-app.zip` ou similaire
   
3. **Extrayez le ZIP**
   - Windows : Clic droit → "Extraire tout"
   - Mac : Double-clic sur le fichier

✅ **TÉLÉCHARGEMENT TERMINÉ !**

---

## 💻 ÉTAPE 2 : INSTALLER NODE.JS (5 minutes)

### **Si vous n'avez pas Node.js installé :**

1. **Allez sur** : https://nodejs.org
2. **Téléchargez** : Version LTS (recommandée)
3. **Installez** : Double-clic et suivre les instructions
4. **Vérifiez** :

```bash
node --version
# Doit afficher : v18.x.x ou supérieur
```

✅ **NODE.JS INSTALLÉ !**

---

## 🚀 ÉTAPE 3 : LANCER L'APP (3 minutes)

### **Ouvrez un terminal :**

- **Windows** : `Win + R` → taper `cmd` → Entrée
- **Mac** : `Cmd + Espace` → taper `terminal` → Entrée

### **Commandes :**

```bash
# 1. Aller dans le dossier (remplacez par votre chemin)
cd chemin/vers/fresh-app

# 2. Installer les dépendances (1-2 minutes)
npm install

# 3. Lancer l'app
npm run dev
```

### **Résultat :**

```
  VITE v6.3.5  ready in 1234 ms

  ➜  Local:   http://localhost:5173/
```

✅ **L'APP EST LANCÉE !**

## 🆘 PROBLÈME ?

### ❌ "npm : commande introuvable"
→ Installer Node.js : https://nodejs.org

### ❌ "Cannot find module"
→ Lancer `npm install`

### ❌ "Port 5173 already in use"
→ Utiliser un autre port : `npm run dev -- --port 3000`

### ❌ Autre problème ?
→ Lire `DEMARRAGE_RAPIDE.md` (section Problèmes courants)
